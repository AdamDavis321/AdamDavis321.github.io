Estimating Scratchpad

The Estimating Scratchpad is a simple web app for estimating software projects. Here is an explanation of the algorithm it uses. This type of calculation can easily be done in a spreadsheet, and that's where I would recommend doing it. I only developed the Estimating Scratchpad to document the estimation process and try out some new ideas for organizing JavaScript applications.

Special Characters

This dynamic table shows the special characters which are defined (named) in HTML. The grouping and notes were added by me. You can select which groups are shown by clicking the check-boxes, and sort the table by clicking on the column headings.

This dynamic table can show the UTF-8 characters from code point U+0000 to U+2FFF.
